# Quiz de francais

![HTML](https://img.shields.io/badge/HTML-CSS-orange) ![Francais](https://img.shields.io/badge/langue-francais-blue)

Outil d'apprentissage interactif du francais - questions de grammaire et vocabulaire dans le navigateur.

## Contenu

- Questions de grammaire
- Vocabulaire
- Corrections avec explications

## Usage

Ouvrir `index.html` dans un navigateur - aucune installation requise.
